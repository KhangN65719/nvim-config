# nvim-config

My personal Neovim config built on [LazyVim](https://lazyvim.org). Feel free to use it as a starting point or inspiration for your own setup.

## Requirements

- Neovim >= 0.9.0
- Git
- A [Nerd Font](https://www.nerdfonts.com/)
- `ripgrep`
- `fd`

## Install

> Back up your existing config first:
> ```bash
> mv ~/.config/nvim ~/.config/nvim.bak
> mv ~/.local/share/nvim ~/.local/share/nvim.bak
> ```
```bash
git clone https://github.com/KhangN65719/nvim-config.git ~/.config/nvim
nvim
```

Lazy will bootstrap itself and install everything on first launch.
